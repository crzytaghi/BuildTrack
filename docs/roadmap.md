# Product Roadmap

## MVP
- Auth: signup, login, logout
- First-login company setup (company name stored locally for now)
- Project CRUD (create + update)
- Task list and expenses (project-scoped)
- Document storage (placeholder for MVP)
- Reports (basic)

## Post-MVP
- Multi-user per company with role-based access
- Subcontractor portal (bids, schedule updates)
- Expense approvals + notifications
- Accounting integrations
- Multi-company support
